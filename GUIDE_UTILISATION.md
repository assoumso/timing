## MODULE 1 : CONNEXION & SÉCURITÉ

### 1. Comment se connecter à l'application
1. Ouvrez votre navigateur (Chrome, Edge, Firefox) et accédez à l'URL de l'application.
2. Sur l'écran de connexion, saisissez votre Identifiant (email) dans le champ prévu.
3. Saisissez votre Mot de passe dans le champ suivant.
4. Cliquez sur le bouton bleu 'Se connecter'.
5. Si vos identifiants sont corrects, vous êtes redirigé vers le Tableau de Bord correspondant à votre rôle.
6. En cas d'erreur, un message rouge s'affiche. Vérifiez l'identifiant et le mot de passe saisis.

### 2. Comment se déconnecter
1. En haut à droite de l'interface, cliquez sur votre nom d'utilisateur ou l'icône de profil.
2. Dans le menu déroulant, cliquez sur 'Se déconnecter'.
3. Vous êtes redirigé vers l'écran de connexion. Votre session est fermée.

## MODULE 2 : ADMINISTRATION & GESTION DES UTILISATEURS

### 1. Comment créer un nouveau compte utilisateur
1. Connectez-vous avec un compte Directeur ou Super Admin.
2. Cliquez sur l'onglet 'Administration' dans la barre de navigation.
3. Allez à la section 'Créer un nouveau compte utilisateur'.
4. Remplissez : Nom complet, Identifiant/Email, Mot de passe.
5. Sélectionnez le Rôle : Super Admin, Directeur, Informaticien ou Professeur.
6. Si c'est un professeur, associez-le à un enseignant de la liste via 'Professeur lié'.
7. Cliquez sur le bouton orange 'Créer le compte utilisateur'.
8. Le compte apparaît dans la liste des comptes ci-dessous.

### 2. Comment attribuer la permission 'Saisie des Moyennes' à un professeur
1. Dans l'onglet Administration, repérez la table 'Comptes utilisateurs enregistrés'.
2. Localisez le compte du professeur concerné dans la liste.
3. Dans la colonne 'Permissions', cochez la case 'Saisie des Moyennes'.
4. La permission est activée instantanément, sans déconnexion requise.
5. Pour révoquer le droit, décochez la même case.

### 3. Comment modifier ou supprimer un compte utilisateur
1. Dans l'onglet Administration, repérez le compte dans la liste.
2. Pour modifier : cliquez sur le bouton Crayon (Modifier) à droite du compte.
3. Mettez à jour les champs souhaités (nom, rôle, mot de passe).
4. Cliquez sur 'Enregistrer les modifications'.
5. Pour supprimer : cliquez sur le bouton rouge Corbeille et confirmez la suppression.

## MODULE 3 : CONFIGURATION DE L'ÉTABLISSEMENT

### 1. Comment configurer les informations de l'école
1. Accédez à l'onglet 'Paramètres' (icône engrenage) dans la navigation.
2. Dans la section 'Informations de l'établissement', modifiez :
  - Nom officiel de l'école
  - Sous-titre ou devise
  - Adresse, Téléphone, Email de contact
  - Nom du Directeur (imprimé sur les bulletins)
  - Année scolaire en cours (ex: 2025-2026)
8. Cliquez sur 'Enregistrer les paramètres' pour sauvegarder.
9. Ces informations apparaissent automatiquement sur tous les bulletins et procès-verbaux imprimés.

## MODULE 4 : GESTION DES CLASSES

### 1. Comment créer une nouvelle classe
1. Allez dans l'onglet 'EDT - Salles & Classes' (ou module Académique).
2. Dans le formulaire 'Ajouter une Classe', saisissez :
  - Identifiant unique (ex: 6A, 5B, 4C, 3D)
  - Nom complet de la classe (ex: Classe de 6ème A)
  - Capacité maximale d'élèves
  - Couleur d'affichage dans l'EDT
  - Professeur Principal (sélectionnez dans la liste des enseignants)
8. Cliquez sur 'Ajouter la Classe'. Elle apparaît dans la grille d'emplois du temps.

### 2. Comment modifier ou supprimer une classe
1. Dans la liste des classes affichées, cliquez sur le bouton Modifier (crayon) de la classe.
2. Modifiez les champs (nom, capacité, professeur principal, couleur).
3. Cliquez sur 'Enregistrer'. Les changements sont reflétés dans tous les modules.
4. Pour supprimer : cliquez sur le bouton rouge Corbeille et confirmez.
5. ATTENTION : La suppression d'une classe retire aussi tous ses cours planifiés.

## MODULE 5 : GESTION DES MATIÈRES & COEFFICIENTS

### 1. Comment ajouter une nouvelle matière
1. Allez dans l'onglet 'Matières' ou 'EDT - Classes & Séries'.
2. Dans le formulaire 'Ajouter une Matière', saisissez :
  - Nom de la matière (ex: Mathématiques, Français, Anglais)
  - Coefficient : poids de la matière dans le calcul des moyennes (ex: 4 pour Math)
  - Cochez 'Langue Vivante 2 (LV2)' si c'est une langue facultative (ex: Espagnol, Allemand)
  - Choisissez une couleur pour l'affichage dans l'EDT
7. Cliquez sur 'Ajouter la Matière'.

### 2. Comment modifier le coefficient d'une matière
1. Dans la liste des matières, cliquez sur le bouton Modifier de la matière.
2. Changez la valeur du Coefficient selon les directives pédagogiques.
3. Cliquez sur 'Enregistrer'. Les bulletins recalculent automatiquement les moyennes.

## MODULE 6 : GESTION DES ENSEIGNANTS

### 1. Comment ajouter un enseignant
1. Allez dans l'onglet 'Enseignants & Équipes'.
2. Dans le formulaire d'ajout, saisissez :
  - Nom et Prénom complets
  - Matière(s) enseignée(s)
  - Téléphone et Email professionnel
  - Statut (Titulaire, Contractuel, Vacataire)
7. Cliquez sur 'Ajouter l'Enseignant'.

### 2. Comment déclarer les indisponibilités d'un enseignant
1. Allez dans l'Espace Enseignant (Portail Professeur).
2. Sélectionnez le professeur dans la liste déroulante en haut.
3. Repérez la grille horaire hebdomadaire affichée.
4. Cliquez directement sur une case de la grille pour la basculer en 'Indisponible' (case grisée avec ❌).
5. Recliquez sur la case pour la remettre en disponible.
6. Les créneaux indisponibles bloquent automatiquement la planification de cours sur ces plages.

### 3. Comment affecter un professeur principal à une classe
1. Allez dans l'onglet de gestion des Classes.
2. Cliquez sur Modifier pour la classe concernée.
3. Dans le champ 'Professeur Principal', sélectionnez l'enseignant dans la liste déroulante.
4. Cliquez sur 'Enregistrer'. Le nom apparaîtra sur tous les bulletins de cette classe.

## MODULE 8 : EMPLOI DU TEMPS (EDT)

### 1. Comment planifier un cours
1. Allez sur l'onglet 'EDT - Planification'.
2. Dans le formulaire à gauche, sélectionnez dans l'ordre :
  1. La Classe (ex: 6ème A)
  2. Le Professeur (ex: M. Koffi)
  3. La Matière (ex: Mathématiques)
  4. La Salle (ex: Salle 101)
  5. Le Jour (ex: Lundi)
  6. Le Créneau horaire (ex: 08:00 - 10:00)
9. Cliquez sur 'Planifier ce cours'.
10. Si aucun conflit n'est détecté, le cours s'affiche dans la grille hebdomadaire.
11. Si un conflit est détecté (prof occupé, salle prise, indispo), une alerte rouge bloque la création.

### 2. Comment supprimer un cours planifié
1. Dans la grille d'EDT, cliquez sur le cours à supprimer.
2. Un panneau de détail s'ouvre à droite ou une boîte de dialogue apparaît.
3. Cliquez sur le bouton rouge 'Supprimer ce cours' et confirmez.
4. Le créneau est immédiatement libéré.

### 3. Comment imprimer l'emploi du temps
1. Dans l'onglet EDT, utilisez les filtres en haut pour choisir la vue :
  - Par Classe : affiche tous les cours de la classe sélectionnée
  - Par Enseignant : affiche le planning personnel du professeur sélectionné
4. Cliquez sur le bouton 'Imprimer' (icône imprimante).
5. La boîte de dialogue d'impression du navigateur s'ouvre. Choisissez 'Enregistrer en PDF' ou imprimez directement.

## MODULE 9 : GESTION DES ÉLÈVES

### 1. Comment inscrire un nouvel élève
1. Allez dans l'onglet 'Élèves' du menu principal.
2. Dans le panneau 'Inscrire un nouvel élève', remplissez :
  - Nom de famille
  - Prénom(s)
  - Genre (Masculin / Féminin)
  - Classe d'affectation (sélectionnez dans la liste des classes)
7. Cliquez sur 'Enregistrer l'inscription'.
8. L'élève apparaît dans le registre de sa classe et dans la liste des bulletins.

### 2. Comment modifier les informations d'un élève
1. Dans la liste des élèves, cliquez sur le bouton Modifier (crayon) à droite de l'élève.
2. Mettez à jour les champs nécessaires (nom, classe, genre).
3. Cliquez sur 'Enregistrer'. Les bulletins et registres sont mis à jour automatiquement.

### 3. Comment changer un élève de classe
1. Cliquez sur Modifier pour l'élève concerné.
2. Changez la 'Classe d'affectation' dans le menu déroulant.
3. Cliquez sur 'Enregistrer'.
4. ATTENTION : Les notes déjà saisies pour l'ancienne classe restent associées à l'élève.

## MODULE 10 : SAISIE DES NOTES & ÉVALUATIONS

### 1. Comment saisir une note via le registre administratif
1. Connectez-vous avec un compte Directeur ou Informaticien.
2. Allez dans l'onglet 'Évaluations & Examens'.
3. Dans le formulaire 'Saisir une Note d'Évaluation' :
  - Sélectionnez l'Élève dans la liste déroulante
  - Sélectionnez la Matière
  - Choisissez le Coefficient de l'évaluation (1, 2, 3 ou 5)
  - Saisissez l'intitulé du devoir (ex: Interrogation Écrite n°1)
  - Entrez la Note sur 20 (ex: 14.5)
9. Cliquez sur 'Soumettre au Registre Électronique'.

### 2. Comment saisir des notes via le portail enseignant
1. Connectez-vous avec un compte Professeur autorisé (permission 'Saisie des Moyennes' active).
2. Allez dans 'Espace Enseignant' > sous-section 'Saisie des Moyennes'.
3. Sélectionnez votre Classe et votre Matière dans les filtres.
4. La liste de vos élèves apparaît.
5. Pour chaque élève, saisissez sa note dans le champ numérique.
6. Renseignez l'intitulé de l'évaluation et le coefficient.
7. Cliquez sur le bouton orange 'Enregistrer toutes les notes saisies'.

### 3. Comment figer (valider) les notes d'une matière
1. Dans la section 'Notes déjà saisies' (côté droit du portail enseignant), vérifiez vos notes.
2. Si tout est correct, cliquez sur le bouton vert '🔒 Valider & Figer'.
3. Confirmez dans la boîte de dialogue qui s'affiche.
4. Les notes passent au statut '🔒 Figé'. Vous ne pouvez plus les modifier.
5. Seul le Directeur ou l'Informaticien peut les déverrouiller en cas d'erreur.

### 4. Comment déverrouiller une note figée (Directeur/Admin uniquement)
1. Connectez-vous avec un compte Directeur ou Super Admin.
2. Accédez au Portail Enseignant, sélectionnez la classe et la matière concernée.
3. Dans la liste des notes saisies, repérez la note marquée '🔒 Figé'.
4. Cliquez sur le bouton bleu '🔓 Déverr.' à droite de la note.
5. Confirmez dans la boîte de dialogue.
6. Le badge '🔒 Figé' disparaît. L'enseignant peut à nouveau modifier la note depuis sa session.

### 5. Comment supprimer une note erronée
1. Seuls le Directeur et l'Informaticien peuvent supprimer des notes.
2. Dans l'onglet Évaluations, repérez la note dans la liste 'Derniers coefficients enregistrés'.
3. Survolez la ligne avec la souris : un bouton rouge Corbeille apparaît.
4. Cliquez sur la Corbeille et confirmez la suppression.
5. Si la note est figée, utilisez d'abord le déverrouillage avant de la supprimer.

## MODULE 11 : PROCÈS-VERBAUX ET BULLETINS

### 1. Comment valider le procès-verbal (PV) d'une classe
1. Vérifiez d'abord que TOUTES les notes de la classe ont été saisies et figées.
2. Allez dans l'onglet 'Évaluations' > sous-onglet 'Procès-Verbaux & Validation'.
3. Sélectionnez la Classe dans la liste déroulante.
4. Consultez la grille de la Natte : vérifiez que chaque matière a une moyenne calculée.
5. Si tout est complet et correct, cliquez sur 'Valider le PV pour cette Classe'.
6. Le statut de la classe passe à 'PV Validé ✅'.
7. Les bulletins de cette classe sont maintenant disponibles pour impression.

### 2. Comment imprimer le bulletin scolaire d'un élève
1. Vérifiez que le PV de la classe a été validé au préalable.
2. Allez dans l'onglet 'Évaluations' > sous-onglet 'Éditeur de Bulletins'.
3. Sélectionnez la Classe et l'Élève dans les menus déroulants.
4. Le bulletin s'affiche avec : toutes les matières, les moyennes, le rang, la mention et les appréciations.
5. Cliquez sur 'Imprimer le Bulletin' (icône imprimante).
6. La fenêtre d'impression système s'ouvre. Sélectionnez 'Enregistrer en PDF' ou imprimez directement.
7. Si le PV n'est pas validé, un message d'avertissement jaune s'affiche et bloque l'impression.

## MODULE 12 : SUIVI DES PRÉSENCES (ABSENCES)

### 1. Comment enregistrer les absences d'un élève
1. Allez dans l'onglet 'Présences & Absences'.
2. Sélectionnez la Classe et la Date du jour.
3. La liste des élèves de la classe s'affiche.
4. Pour chaque élève absent, cliquez sur la case correspondante pour la cocher.
5. Indiquez si l'absence est Justifiée ou Non Justifiée.
6. Cliquez sur 'Enregistrer l'appel' pour sauvegarder.

### 2. Comment consulter l'historique des absences d'un élève
1. Dans l'onglet Présences, allez dans le sous-onglet 'Historique'.
2. Filtrez par Élève et par Période (mois ou trimestre).
3. Le tableau affiche toutes les dates d'absence avec le statut (justifiée/non justifiée).
4. Le total d'heures d'absence est calculé automatiquement.

## MODULE 13 : FINANCES & ÉCOLAGES

### 1. Comment enregistrer un paiement de scolarité
1. Allez dans l'onglet 'Finances & Écolages'.
2. Dans la section 'Nouvelle Transaction', sélectionnez l'Élève concerné.
3. Renseignez :
  - Montant versé (ex: 75 000 FCFA)
  - Motif : Inscription, Scolarité 1er/2ème/3ème Trimestre, Fournitures, etc.
  - Mode de règlement : Espèces, Chèque, Orange Money, Wave, MTN Money
  - Date du versement
8. Cliquez sur 'Enregistrer la transaction'.
9. Le solde restant de l'élève est recalculé automatiquement.

### 2. Comment générer et imprimer un reçu de paiement
1. Après avoir enregistré un paiement, repérez-le dans la table des transactions.
2. Cliquez sur le bouton 'Reçu PDF' à droite de la transaction.
3. Un reçu officiel au format PDF est généré avec :
  - Le nom de l'établissement et le logo
  - Le nom de l'élève et sa classe
  - Le montant payé, le motif et la date
  - Un numéro de reçu unique
8. Cliquez sur 'Télécharger' ou 'Imprimer' selon votre besoin.

### 3. Comment envoyer un reçu par WhatsApp ou SMS
1. Après avoir généré le reçu, cliquez sur le bouton vert 'WhatsApp'.
2. Votre application WhatsApp s'ouvre avec un message pré-rempli contenant les détails du paiement.
3. Sélectionnez ou saisissez le numéro du parent/tuteur de l'élève.
4. Cliquez sur Envoyer. Le parent reçoit instantanément la confirmation de paiement.

## MODULE 14 : RAPPORTS & STATISTIQUES

### 1. Comment générer un rapport de classe
1. Allez dans l'onglet 'Rapports & Statistiques'.
2. Sélectionnez le type de rapport : Résultats scolaires, Présences, Financier.
3. Choisissez la Classe et la Période (Trimestre 1, 2 ou 3, ou Annuel).
4. Cliquez sur 'Générer le Rapport'.
5. Le rapport s'affiche avec des graphiques et des tableaux récapitulatifs.
6. Cliquez sur 'Exporter en PDF' ou 'Exporter en Excel' pour le sauvegarder.

### 2. Comment consulter les statistiques du tableau de bord
1. Accédez à l'onglet 'Tableau de Bord' (page d'accueil après connexion).
2. Les cartes colorées affichent en temps réel :
  - Effectif total d'élèves inscrits
  - Nombre de cours planifiés cette semaine
  - Taux de remplissage des salles
  - Nombre de professeurs actifs
7. Consultez les graphiques de répartition par classe et par genre.

## MODULE 15 : SÉCURITÉ & PROTECTION DES DONNÉES

### 1. Comprendre les protections de sécurité de l'application
1. L'application interdit le clic droit sur les pages pour protéger le code source.
2. La touche F12 (outils développeur) est bloquée pour les utilisateurs non-admin.
3. Le raccourci Ctrl+S (sauvegarde page) et Ctrl+U (voir source) sont désactivés.
4. L'application ne fonctionne que sur les domaines autorisés (localhost, vercel.app).
5. Si l'URL n'est pas reconnue, un écran de verrouillage de domaine s'affiche.

### 2. Comment gérer les permissions d'accès aux onglets
1. Connectez-vous en tant que Directeur ou Super Admin.
2. Allez dans l'onglet Administration.
3. Pour chaque utilisateur, cochez ou décochez les onglets auxquels il a accès dans la colonne 'Pages autorisées'.
4. Les modifications sont instantanées.
5. Un utilisateur sans permission sur un onglet ne verra pas cet onglet dans son menu de navigation.
